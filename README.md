# Hospital-Management-System-CPP
A C++ console-based Hospital Management System for managing patient records, appointments, and hospital operations using OOP concepts.

## Features
1. Add new patient records
2. View patient details
3. Search patients by ID
4. Delete patient records
5. Menu-driven user interface
6. Data organization using Object-Oriented Programming concepts

## Technologies Used
1. C++
2. Object-Oriented Programming (OOP)
3. Classes and Objects
4. Functions
5. Arrays
6. Conditional Statements
7. Loops

## How to Use
1. Launch the program.
2. Select an option from the menu.
3. Add, view, search, or delete patient records.
4. Manage hospital information.
5. Exit the system when finished.

## Learning Outcomes
1. Applied Object-Oriented Programming concepts in a real-world project.
2. Learned data management and record handling.
3. Improved problem-solving and program design skills.
4. Practiced menu-driven application development.
5. Gained experience in organizing large C++ programs.
